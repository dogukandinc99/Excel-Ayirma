# Excel Ayırma

Bu program excelde belirli bir sütuna göre farklı sayfalara ayırıyor. Ayrılan sayfalarda belirli bir sütuna göre aralarına boşluk ekliyor. Bu program sayesinde excelde ayırmak istediğimiz satırları kolay bir şekilde ayırabiliyoruz. Hesaplamalarda istediğimiz değerler ayrı ayrı sayfalarda düzenli bieşekilde bulunacağından dolayı daha hızlı işlemlerimizi yapabiliyoruz.
